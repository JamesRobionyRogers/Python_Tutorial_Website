Functions in Python

Medium - Callable Blocks of Code

The Basics

Creating a Function

A function is a block of code that will only run once called upon. You can pass data such as strings, integers, floats, booliens and more through a function.

As a result of this, functions are able to also return data.


So how do we create a function?

Functions are created by using def followed by the name of the function. As demonstrated below:

def greeting():
   print(f"Hello, welcome to the functions tutorial")

Calling a Function

Calling a function is as easy as adding two round brackets to the end of a word.

def greeting():
   print(f"Hello, welcome to the functions tutorial")

greeting()
>>>  "Hello, welcome to the functions tutorial"

Passing Arguments Through a Function

As mentioned above, you are able to pass infomation through functions, these are known as arguments.

Arguments are specified in between the round brackets when defining the function. You are allowed to pass as many arguments as you want, all you need to do is make sure they are seperated by commas. Here's an example:

def greeting(name, age):
   print(f"Hello, {name}. Welcome to the functions tutorial")
   print(f"From the data I've gathered, you are {age} years old")

greeting('James', 17)
>>>  "Hello, James. Welcome to the functions tutorial"
>>>  "From the data I've gathered, you are 17 years old"

Here's another example of passing data through a function:

def multiply_by_pi(number):
   pi = 3.14159265359
  # by using the return method, the function then sets the variable to that value
   return number * pi

result = multiply_by_pi(15)
print(result)
>>>  47.1238898038
