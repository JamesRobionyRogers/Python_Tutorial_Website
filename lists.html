Lists in Python

Easy - Medium - Iterating through lists, Modifying lists & 2D lists

The Basics

A list in Python is a sequence of items. We write this sequence in between square brackets, separating the items using a comma. As shown below.

The items in a list can be of any type (including lists themselves). So, the following are all valid lists.

empty_list = []  # here is an empty list being assigned to a variable
list_with_numbers = [1, 2, 3]  # as you can see here we are separating the items using commas
another_list = [ [19, 'September'], [20, 'September'] ]  # here is a multi dimensional list

The order of items in a list is important, as is the number of items in a list.

We can:

  • Add new items to a list
  • Remove items from a list
  • Change the order of items in the list

Think of a list like a variable, except that it can have multiple values stored inside it. For example a variable called name might have the value 'Hone' stored in it, a list called names might have the values 'Hone', 'Kate', 'Fariha' and 'Yang' stored in it.

name = 'Hone'
names = ['Hone', 'Kate', 'Fariha', 'Yang']

As shown in the first example, lists can store data of different types as well. We could store some text (a string) as well as some integers, floats and boolean values.

my_list = ['Pizza', 3.14, 8, True]

Accessing values in a list

One of the most important things that we need to do with a list is to access the individual items in it. We use the index to do this. If l is a list and l is a number then l[i] is an item of l.

Each item in a list has an index, or a number associated with it. The numbers are sequential with the first item in the list being associated with the number 0. This means that if there are 8 items in a list the number associated with the last item will be 7. Confusing huh?

So say we wanted to print Kate's name from the names list. We know her name is in the second position in the list, so starting from 0 the second number is 1. So we would write:

>>>  names = ['Hone', 'Kate', 'Fariha', 'Yang']
>>>  print(names[1])
>>>  'Kate'
>>>  print(names[-1])  # you can also start from the back of the list by using a negitive index value
>>>  'Yang'

The square brackets after the list name contain the index (or location) in the list that we are accessing.

Traversing a List

Often we will want to process the elements in a list one-by-one building up a value as we go along (called a fold). The code will typically look like this:

# there are two ways of doing this, the more complicated way:
big_five = ['elephant', 'lion', 'rhino', 'leopard', 'buffalo']

for i in range(len(big_five)):
   # uses the index value of the list to iterate through the list
   print(big_five[i])

# if you wanted an eaiser way, here it is
for animal in big_five:
   print(animal)

Adding & Removing From a List

Modifying a Value in a List

If we know the index of the item we want to edit in the list we can change its value, just as we would do with a variable.

# declares a list called 'ducks' consisting of mixed datatypes
>>>  ducks = ["Huey", True, "Duey", False]
# change the value of the item in the list 'ducks' with the index of 2
>>>  ducks[2] = "Dewey"
>>>  print(ducks)
["Huey", True, "Dewey", False]

Adding new values to a list

We can add a new value to the end of our list by using the append() function

# declares a list called 'ducks' consisting of mixed datatypes
>>>  ducks = ["Huey", True, "Dewey", False]
# adds an item to the end of the list
>>>  ducks.append("Louie")
>>>  print(ducks)
["Huey", True, "Dewey", False, "Louie"]

Inserting a new value into the list

We can also choose where we can add an item into a list by using the insert() function. insert() takes 2 arguments, the index position and the value, in that order. Everything else in the list will shift as the list expands.

#declares a list called 'ducks' consisting of mixed datatypes
>>>  ducks = ["Huey", True, "Dewey", False, "Louie"]
#adds an item to a specific index value inside the list
>>>  ducks.insert(0, "Donald")
>>>  print(ducks)
["Donald", "Huey", True, "Dewey", False, "Louie"]

Removing items from a list

If we know the index value of the item we are looking to remove we can use the del() function. The list will contract to account for the removed item.

#declares a list called 'ducks' consisting of mixed datatypes
>>>  ducks = ["Donald", "Huey", True, "Dewey", False, "Louie"]
#deletes an item from a specific index value in the list
>>>  del ducks[2]
>>>  print(ducks)
["Donald", "Huey", "Dewey", False, "Louie"]

Finally we can remove an item from the list by its value. Warning, this removes the first matching value.

#declares a list called 'ducks' of mixed datatypes
>>>  ducks = ["Donald", "Huey", True, "Dewey", True, "Louie"]
#removes the first instance of a value in a list
>>>  ducks.remove(True)
>>>  print(ducks)
["Donald", "Huey", "Dewey", True, "Louie"]

2D Lists

A list is heterogeneous. This means that we can put any type of data into a list: strings, integers, floats and Booleans. We can also put lists inside lists! This makes a list 2-Dimensional.

To create a 2D list you would write the notation like:

# 2D lists are lists that are inside another list, like so
movies = [['The Hobbit', 182], ['Lord of the Rings', 228], ["Harry Potter", 159]]

Adding to a 2D List

Adding or inserting into a 2D list is just as simple as it is with a 1D list. Here's an example.

movies[0].append('7.8 ★')  # appending to 'The Hobbit' list
print(movies)
>>>  [['The Hobbit', 182, '7.8 ★'] ... ]
movies[1].insert(1, '8.8 ★')  # inserting into 'Lord of the Rings' list
print(movies)
>>>  [ ... ['Lord of the Rings', '8.8 ★', 228] ... ]

Removing Items From a 2D List

Removing is items is also other same, we can pop() or remove()

movies[0].remove(182)  # removing from 'The Hobbit' list
print(movies)
>>>  [['The Hobbit', '7.8 ★'] ... ]
movies[1].pop(0)
print(movies)
>>>  [ ... ['8.8 ★', 228] ... ]

Accessing Items Within a 2D List

If we want to access a value in our list be careful! You don't write it with a square bracket inside a square bracket list how you create your 2D list. Instead you write it next to each other with the first index referring to the first index of the list and the second the second.

movies = [['The Hobbit', 182], ['Lord of the Rings', 228], ["Harry Potter", 159]]

# accessing a 2D list is simple
print(f'Title: {movies[0][0]}   Runtime: {movies[0][1]}')
>>>  'Title: The Hobbit   Runtime: 182'
print(f'Title: {movies[2][0]}   Runtime: {movies[2][1]}')
>>>  'Title: Harry Potter   Runtime: 159'

# Technically you could just keep going on with a 3D list etc.

Filling a 2D List

If we wanted to fill our 2D list we would need a nested loop. Additionally inside each loop we would need to create a new list.

students = []
names = ['James', 'Daniel', 'Callum', 'George']
ages = [17, 16, 14, 10]

for index_value in range(len(names)):
   students.append([names[index_value], ages[index_value]])

print(students)
>>>  [['James', 17], ['Daniel', 16], ['Callum', 14], ['George', 10]]
