Loops in Python

Easy - For Loops, While Loops

For Loops

For loops are used to iterate through sequences. These sequences can come in the form of: lists, tuples, dictionaries, sets or strings.

Using a for loop we can execute a block of code for every iteration of a sequence.


Here are a couple of basic examples of for loops in action:

# This for loop simply iterates through, printing out each item in the list defigned below
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
   print(fruit)

# This for loop is known as a counting loop, as it iterates from the numbers 1, through to 5
for counting in range(1, 5):
   print(counting)

Another example of for loops being used are in nested loops. This is when you put a for loop inside another for loop. Hence the name "nested".

origins = ["tree", "bush", "plant"]
fruits = ["pear", "blackberry", "strawberry"]

for fruit in fruits:
   for origin in origins:  # loop nested inside another loop
      print(f"An {fruit} grows on a {origin}")

While Loops

As for a while loop, here are the basics to that.

A while loop will run the block of code untill as long as a contion is met. A very simple example of this is an infinate while loop, it will continue running the block of code and will never stop since the condition is always being met. 

# the condition "True" is always true and cannot be changed. Hence making it an infiniate loop
while True:
   print("Infinate loop!")

Here is a more common example of a while loop.

This loop will continue to execute the block of code while the condition of age if between the values of 13 and 19. Once age exceded this, the condition is no longer met, ending the loop.

age = 13
while age >= 13 and age <= 19: # here is the conditional statement.
   # while the age is between the values of 13 and 19 everything in the loop will run
   print("I am a teenager ", age)
   age += 1

While Else Loops

Building upon the while loops, you can also add an else clause as you would do with an if statement. The else clause will run once the while loop is finished.

# here is an example of a while else statement
iteration = 0
while iteration != 3:
   print(iteration, "Inside while loop")
   iteration += 1
else:  # runs once the while loop finishes
   print("Inside else clause")
